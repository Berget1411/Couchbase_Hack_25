### FastAPI middleware for comprehensive request/response logging.

## Installation

```bash
pip install -e .
```

## Usage

```python
from fastapi import FastAPI
from smartpylogger import LoggingMiddleware

app = FastAPI()

# Add your middleware
app.add_middleware(
    LoggingMiddleware,
    api_key="YOUR_API_KEY",
    allowed_origins=["IP", ...]
    )
```
